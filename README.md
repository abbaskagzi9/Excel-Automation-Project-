# 📊 Excel Payroll Automation System with Automated Payslip Generation

## 📌 Overview

The **Excel Payroll Automation System** is a Microsoft Excel-based project developed using **Visual Basic for Applications (VBA)** and **Excel Macros**. It automates the payroll process by calculating employee salaries, applying allowances and deductions, and generating professional payslips automatically.

The objective of this project is to reduce manual effort, improve accuracy, and demonstrate how Microsoft Excel can be used to automate real-world HR and Finance operations.

# ⚙️ How the System Works

### Step 1

Open the Excel workbook and enable Macros.

### Step 2

Enter employee details including:

* Employee ID
* Employee Name
* Department
* Basic Salary
* HRA
* Allowances
* Deductions

### Step 3

Click the **Generate Payroll** button.

The VBA Macro automatically:

* Calculates Gross Salary
* Calculates Total Deductions
* Calculates Net Salary
* Updates Payroll Records

### Step 4

Click **Generate Payslip**.

The macro automatically creates a professional payslip containing:

* Employee Information
* Salary Breakdown
* Allowances
* Deductions
* Net Salary
* Payroll Summary
